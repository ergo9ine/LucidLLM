# LucidLLM 기능 로드맵

## 추론 및 모델

- [ ] **스트리밍 응답** — 토큰 단위로 실시간 출력하여 응답 대기 시간 체감 개선
- [ ] **멀티모달 입력** — 이미지 첨부 후 Vision 모델로 분석 (LLaVA, Moondream 등)
- [ ] **모델 비교 모드** — 동일 프롬프트를 두 모델에 동시 전송하여 응답 품질 비교
- [ ] **양자화 자동 추천** — 사용자 기기의 VRAM/메모리에 맞는 최적 양자화 레벨 자동 제안
- [ ] **모델 프리셋** — 인기 모델 목록을 원클릭으로 다운로드할 수 있는 큐레이션 프리셋
- [ ] **RAG 지원** — 로컬 문서(PDF, TXT, MD)를 업로드하여 검색 증강 생성 수행
- [ ] **Function Calling** — 도구 사용 인터페이스로 계산기, 웹 검색 등 외부 기능 연동

## 채팅 UX

- [ ] **Markdown 렌더링** — 코드 블록 구문 강조, 테이블, LaTeX 수식 렌더링
- [ ] **메시지 편집/재생성** — 전송된 메시지 수정 후 재생성 기능
- [ ] **대화 분기(Fork)** — 특정 메시지 시점에서 대화를 분기하여 다른 방향으로 탐색
- [ ] **대화 검색** — 전체 채팅 기록에서 키워드 기반 검색
- [ ] **대화 내보내기** — JSON, Markdown, TXT 형식으로 채팅 기록 다운로드
- [ ] **드래그 앤 드롭 파일 첨부** — 채팅 입력란에 파일을 드롭하여 컨텍스트로 첨부
- [ ] **음성 입력/출력** — Web Speech API를 활용한 STT 입력 및 TTS 응답 읽기

## 성능 및 인프라

- [ ] **WebGPU 메모리 모니터링** — GPU 메모리 사용량 실시간 표시 및 OOM 사전 경고
- [ ] **추론 벤치마크** — 모델별 tokens/sec, 첫 토큰 지연시간(TTFT), 메모리 사용량 측정 대시보드
- [ ] **백그라운드 다운로드** — 모델 다운로드 중에도 기존 모델로 채팅 계속 가능
- [ ] **SharedArrayBuffer 지원** — COOP/COEP 헤더 설정으로 멀티스레드 WASM 추론 활성화
- [ ] **IndexedDB 폴백** — OPFS 미지원 브라우저를 위한 IndexedDB 기반 모델 캐시 폴백

## PWA 및 배포

- [ ] **Web App Manifest** — `manifest.json` 추가로 완전한 PWA 설치 경험 제공
- [ ] **푸시 알림** — 장시간 추론 완료 시 백그라운드 알림
- [ ] **캐시 버전 관리** — SW 캐시 이름에 빌드 해시를 포함하여 업데이트 시 자동 캐시 무효화
- [ ] **GitHub Actions CI** — PR마다 자동 린트, 테스트, Lighthouse 감사 실행

## 보안 및 프라이버시

- [ ] **CSP 헤더** — Content Security Policy로 XSS 방어 강화
- [ ] **로컬 암호화 저장** — 채팅 기록을 OPFS에 저장할 때 선택적 로컬 암호화 적용
- [ ] **자동 데이터 만료** — 설정 가능한 기간 이후 오래된 채팅 세션 자동 삭제

## 접근성 및 UI

- [ ] **키보드 단축키** — 새 채팅(`Ctrl+N`), 설정(`Ctrl+,`), 전송(`Ctrl+Enter`) 등
- [ ] **고대비 테마** — 시각 장애 사용자를 위한 WCAG AAA 준수 고대비 테마
- [ ] **폰트 크기 조절** — UI 전체 폰트 크기를 사용자가 조절할 수 있는 슬라이더
- [ ] **사이드바 레이아웃** — 넓은 화면에서 대화 목록을 사이드바로 분리하는 레이아웃 옵션

## 통합 및 확장

- [ ] **플러그인 시스템** — 사용자 커뮤니티가 기능을 확장할 수 있는 플러그인 아키텍처
- [ ] **OpenAI API 호환 엔드포인트** — 외부 도구에서 LucidLLM을 로컬 API 서버처럼 사용
- [ ] **프롬프트 템플릿 라이브러리** — 자주 쓰는 프롬프트를 저장/공유할 수 있는 템플릿 관리
- [ ] **Chrome 확장 프로그램** — 웹 페이지 텍스트 선택 후 LucidLLM으로 바로 분석
